# Quickstart

## Pré-requisitos

- Node.js 22+
- npm 10+
- Git
- Bruno instalado

## Instalação

```bash
npm install
```

### Validar estrutura do repositório
```bash
npm run check:repo
```

### Listar trilhas
```bash
npm run list:trails
```

### Estrutura geral

- apps/: trilhas organizadas por tema

- packages/: configs e templates compartilhados

- tools/: scripts internos e base para o Codex

- docs/: documentação
